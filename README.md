# be-spring-cafe
## URL
| URI                           | 기능          |
| ----------------------------- | ----------- |
| `GET /users`                  | 사용자 목록      |
| `POST /users`                 | 회원 가입 기능    |
| `GET /users/{userId}`         | 회원 프로필 조회   |
| `GET /questions`              | 글쓰기 페이지     |
| `POST /questions`             | 글쓰기 기능      |
| `GET /questions/{questionId}` | 게시글 상세 보기   |
| `GET /users/{userId}/form`    | 회원정보 수정 페이지 |
| `PUT /users/{userId}`         | 회원정보 수정     |
