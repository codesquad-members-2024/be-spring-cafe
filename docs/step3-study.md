## 부모 클래스 메서드의 반환타입이 Collection일 때, 하위 클래스에서 Override하면서 반환타입을 List로 변경해도 문제 없을까?
- 자식 클래스에서 부모 클래스 메서드를 오버라이드 할 때 반환 타입을 더 구체적인 타입으로 변경하는 것은 허용됨
- 자바의 다형성에 기반
- 서브 타입 규칙에 따라 자식 클래스의 반환 타입은 부모 클래스 반환 타입의 서브 타입이어야 한다
